# java-starter


### To build the app

Type this command into the terminal

    ./gradlew build
    
    
### To run the app

This command:


     ./gradlew run

## API


### To get a list of all people:


Go to: [http://localhost:8080/](http://localhost:8080/)


### To get a person by ID:


Go to: [http://localhost:8080/{id}](http://localhost:8080/1)
